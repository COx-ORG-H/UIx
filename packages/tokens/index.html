Data table

Columns are sized by class, not hand-coded px. In fixed layout (.uix-table--fixed) the Subject column flexes to fill space with a min-width floor and truncates to one line (.uix-col--flex .uix-col--truncate); every other column snaps to a width tier (.uix-col--w-*), and the pinned first column stays put as the table scrolls sideways. Row click opens the peek. Hover any row to reveal ↗ in the ID cell — click it to open the full record directly without the peek. ↑ ↓ navigates records within the peek.

Workflows & pipelines

Diagram primitives for visualizing a process — a .uix-node is the boxed step; .uix-flow arranges nodes and draws the connecting edges. Compose them into linear flows, branches, loops, mind maps, or a builder canvas. Edges are decorative SVG (aria-hidden); state is carried by a text label, never colour alone. For lightweight cases prefer the primitives you already have — .uix-steps, .uix-pipeline, .uix-timeline, .uix-kanban.

Icons

Icons are lucide glyphs, inline SVG, drawn in currentColor at stroke-width:2 with round caps. Size them with --uix-icon-sm (16) / --uix-icon-md (20) / --uix-icon-lg (24) via the icon(name, size) helper in assets/icons.js.

Accessibility
Decorative icons get aria-hidden="true" (the icon() helper adds it). An icon that is a control's only label needs an aria-label on the control (or a visible tooltip). Never rely on an icon alone to carry meaning.

Prose card

Resolving VPN disconnects

When users on the EU gateway report random disconnects, the cause is almost always a stale session table after a config push. Follow these steps before escalating.

Steps

  1. Roll back the most recent gateway config.
  2. Flush the session table with vpnctl flush --region eu.
  3. Watch reconnect rates for 15 minutes.
If disconnects persist after a flush, the issue is upstream — open a problem ticket and link this article.

See the runbook for the full escalation path.

Note callouts
Note. Config pushes take up to 60 seconds to propagate across all regions.
Resolved. Reconnect rate is back to baseline.
Heads up. Flushing sessions will sign out active users.
Do not restart the gateway during business hours.
